## 🚀 Quick Start

1. **Clone & Install**:
   ```bash
   git clone https://github.com/the-vibe-coder-69/www.git
   cd www
   npm install
   ```

2. **Run Dev Server**:
   ```bash
   npm start
   ```

3. **Run Visual Tests**:
   ```bash
   npm run qa:update-screens
   ```
